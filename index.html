Управление инстансом и отправкой сообщений

Подключите ваш инстанс, получите его настройки и состояние, затем отправьте текстовое сообщение или файл по URL. Ответ каждого метода отображается справа в режиме только для чтения.

Параметры подключения

Дополнительные параметры

Адрес API подбирается автоматически. При необходимости можно указать ваш `apiUrl` из личного кабинета вручную.

getSettings

Получить текущие настройки инстанса.

getStateInstance

Проверить текущее состояние инстанса.

sendMessage

Отправить текстовое сообщение в личный чат или группу.

Поддерживаются личные и групповые чаты. Максимальная длина текста: `20000` символов.

Расширенные параметры sendMessage
customPreview

sendFileByUrl

Отправить файл по прямой ссылке.

Используйте прямую ссылку на файл. `fileName` должен содержать расширение, а `caption` может быть длиной до `20000` символов.
